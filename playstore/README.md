# ReceiptVault Play Store Prep

This folder contains the first Play Store listing package for ReceiptVault.

The app package name is:

```text
com.corsairlabs.receiptvault
```

Suggested Play Console setup:

- App name: ReceiptVault
- Default language: English (United States)
- App type: App
- Category: Productivity or Finance
- Free or paid: Free, with in-app subscription later
- Managed publishing: On

The Play Console app itself still needs to be created manually inside the
Corsair Labs Google Play Console account. After the app exists, add Google Play
Billing subscription product IDs and wire them into the Plus screen.
